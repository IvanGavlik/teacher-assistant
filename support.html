Changelog

Version history

Every release, from the current version back to where we started.

v1.1.0 Apr 2026 Latest

Quiz & Homework slides + slide preview

  • Added Quiz slide type — generates multiple choice questions directly as PowerPoint slides
  • Added Homework slide type — auto-generates structured take-home assignment slides
  • New slide preview panel — review and navigate all generated slides before inserting into your presentation
  • Improved error messages for quota limits, connection issues, and authentication failures
  • General stability improvements to the PowerPoint Add-in
v1.0.0 Mar 2026

Initial release

  • Launched PowerPoint Add-in with AI-powered slide generation via real-time WebSocket streaming
  • Vocabulary slide type — word, definition, translation, and example sentence per slide
  • Grammar slide type — rule explanation with structured examples
  • Reading slide type — passage with comprehension content
  • CEFR level support (A1–C2) across all content types
  • Powered by GPT-4 via a dedicated Clojure backend
